README
===========================
这是一个用Latex写的教案模板，基于 [xecjk-template](https://github.com/xiaohanyu/xecjk-template) 中文模板，
本教案模板符合本院的格式要求，首页采用表格制定格式，然后用制定指令替换里面的内容，
教案内容页用 TiKz 画了个带旁注的边框，制定了首页与教案内容的页眉，并对 Latex 默认的格式进行了指定。
